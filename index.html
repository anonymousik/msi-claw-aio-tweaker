Termux Windows

Użyj swojego telefonu z systemem Android jako zewnętrznego monitora diagnostycznego. Poniższy generator stworzy bezpieczny skrypt Bash, który połączy się z Twoim MSI Claw przez SSH i pobierze dane telemetryczne (Bateria, Temp, RAM) w czasie rzeczywistym.

1. Konfiguracja Połączenia

Jak skonfigurować Windows (Wymagane)
1. Otwórz Ustawienia -> Aplikacje -> Funkcje Opcjonalne.
2. Zainstaluj Serwer OpenSSH.
3. Otwórz Usługi (services.msc) i ustaw "OpenSSH SSH Server" na Automatyczny i uruchom go.
4. Sprawdź IP wpisując ipconfig w CMD.
Wymagania Termux (Android)
1. Pobierz Termux (F-Droid lub GitHub).
2. Wpisz: pkg update && pkg install openssh
